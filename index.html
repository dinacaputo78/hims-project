Data Dictionary

Core Definitions (NCAA-inspired)

Athlete Exposure (AE): One athlete participating in one training or competition session where exposed to risk, regardless of duration.

Time-Loss: Time between injury and return to participation, classified by severity in days.

Mandatory Minimum Dataset

  • Event type (training/competition)
  • Number of athletes exposed
  • Anatomical site
  • Injury type
  • Mechanism
  • Recurrence status
  • Time-loss (days)

Time-Loss Severity Classification

  • Minimal: 1-3 days
  • Mild: 4-7 days
  • Moderate: 8-28 days
  • Severe: >28 days
